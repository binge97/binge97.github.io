# 政治复习目标

建议目标：75+

# 问题一：为什么要 75 分 +？

  • 假设你们英语达到上岸同学的平均水平（65 分），那么政治至少 75 分，才能保证两门公共课 140 分。如果你的目标是 380+，那么你专业课就是需要考到 240+。
  • 好学生一定会有好的英语和专业课能力，但政治上大家起点相对还算一致。因此，我们反而要在这一科目上超越其他人，建立起优势。如果你超越了 75 分，那么不仅是超越了别人，更是超越了自己。
  • 总分不变的情况下，政治多一分，就意味着英语和专业课你就多一分的容错，所以政治复习的越好，其他科目就越安全，
  • 政治是第一门考试，虽说大家政治拉不开分数，但是你做的顺利，心态就好，有利于后续的考试。

# 问题二：75 + 难吗？

70 分不难需要努力，75 需要努力 + 动脑，75 + 就是完全的很用心，超越了绝大多数 985 考生。登山难吗？难，但是你知道一步步走，总有上去的一天。75 + 也是一样的。

# 问题三：我能做到吗？

我可以保证如果你 100% 执行好下面说的每一步，肯定是 80 + 的水平。我那年政治成绩 75 分，我自我感觉我做到了 70%。所以下面的经验给你，你再结合自己的情况，合理调整，用心去学习去感受，结果好不好，取决于自己的创造。
下面就给出我具体的复习规划和经验。

# 前言

# 复习要用到的资料

政治资料贼多，每个老师都有其擅长的地方。因此我们没必要从头到尾跟着同一个老师，而是各取所长，有主有次，下面是推荐的组合搭配

  • 教材：徐涛核心考案 + 肖秀荣风中劲草 两本足够
  • 习题集：肖秀荣 1000 题 一本足够
  • 辅助资料： 王吉掌中宝
  • 押题卷：肖四肖八（11 月中下旬购买）
  • 网课：以徐涛为主（风趣幽默，深入浅出）
  • 小程序：政治刷题小程序
  • 公众号：考研政治徐涛
  • A4 纸或笔记本或电脑笔记

除了以上书籍，其他的都不需要买，因为买了也看不完。如果你们想购买什么其他资料，建议先在群里咨询。

# 复习资料的特点

  • 网课：不看基础班时间太长，直接从强化班开始看，强化班就是基础班。冲刺班和押题班更重要，冲刺班都是重点知识，押题班讲了很多做题思路和技巧。所以我们丢弃了不重要的基础班，是为了有更多时间看冲刺和押题。
  • 教材：其实都差不多，核心考案很厚，只有第一遍复习的时候可以当教材看看。后面的话看知识更凝练的疾风劲草，再后面看更加凝练的王吉掌中宝（我后面两天看一遍）

# 复习分为哪几个阶段？

  • 基础阶段（3-4 月）：所有课程过一遍，做一做对应的选择题进行练习，目标是建立知识基础，知道考研政治知识体系、重难点和考试题型。
  • 强化阶段（5-6 月）：按照章节做 1000 题，哪一题不会就去复习对应的章节，目标是检测自己基础哪里不牢固，强化对知识点的理解。
  • 提分阶段（7-9 月）：第二遍刷题，同时看一些做题课 / 技巧课，目标是进一步强化知识，学会做题。
  • 定型阶段（10 月）：整理错题，系统性的记忆选择题考点，坚持刷题；另一方面开始背诵大题
  • 冲刺阶段（11-12 月）：主要是背诵大题，同时对选择题进行查漏补缺，做真题和押题卷。准备去考试。

# 政治学科的特点

学政治首先知道什么是政治，为什么学政治？ 政治包含马原，毛中特，近代史，思修，形势政策与当代政治，一个合格的研究生当然要了解这些。对于考试来讲，马原，毛中特，近代史是重点，后面的最后两个月准备一下就行了。

  • 大题：前期完全不用背，10 月份跟徐涛简单背一背，记不牢也无所谓。肖四出来之后开始背大题，基本上都是原题。不必担心。
  • 选择题：知识点又多又碎，和英语单词一样的性质。所以我不推荐死记硬背，记得快忘得快，有什么用。而是应该每天都看，一年下来各个知识点反复看个十几遍，自然了然于胸了。看的时候用自己的理解方式去记忆，用口诀，用关键词去记忆。要区分他和大题记忆的区别。大题要求默写的，选择题理解就行。另外记忆不是目的，会做题才是最终目标，所以在准备选择题知识点时，要多想想这个知识点如果让你出选择题你怎么考？这就是站在出题人角度。

最后送你一句话，全国考生大题都背的肖四肖八，分数不相上下，因此得选择题者得天下。选择题 40 分，总分就可以 75 + 了。

# 各阶段具体做法

# 基础阶段（3-4 月）：小白初窥门径

0 基础到 70+，做好这一步，一切就都有可能。

  • 用书：核心考案 + 徐涛 24 年强化班（因为 25 年强化课还没更新出来）
  • 目标：建立知识基础，知道考研政治知识体系、重难点和考试题型。
  • 策略：要有大局观，复习要快
  • 复习时长：一定不超过 1.5h

网课大约共 52 节课，平均每节课约 1.5h。每天一节课，截至 4 月底一定可以看的完一遍。

  • 按照 马原 -> 毛中特 -> 近代史 的顺序进行复习
  • 以看网课为主，然后在核心考案上做下简单的标记 / 笔记（注意，不需要都记，记下重要的和不会的，贪多 = 0）
  • 建议 1.5-2 倍速观看，难懂的地方可以多看两遍，或者找其他机构老师的讲解作为辅助理解（比如腿姐）
  • 其实就马原比较难懂，剩下两门课跟看故事一样过一遍就行

总结：这个阶段不需要刻意记忆，而重在理解知识点。从整体上知道知识的框架，以及不同模块的重难点就好了。哪天有多余的时间，就可以往前赶任务。

# 强化阶段（5-6 月）：准备登堂入室

有了前面两个月的基础铺垫，到了这个阶段就是开始真正的进入复习了。要时刻知道自己在学什么，要经常思考自己会不会，不要只管做题或看视频。脑子的清醒度 = 复习效果。

  • 用书： 肖 1000 题 + 核心考案 + 徐涛 25 年强化班
  • 目标：检测自己基础哪里不牢固，强化对知识点的理解。
  • 策略：好好做题，用心做好整理，以点入面
  • 复习时长：一定不超过 1.5h

1000/60=16.6，所以每天需要做 20 题，多余的时间作为容错。

  • 还是按照 马原 -> 毛中特 -> 近代史 的顺序进行做题，每天 20 道题。答案写在空白纸上，不要写在习题册上（后面还要再做一遍）。
  • 做完看答案，对的题看答案思路和自己一不一样。错的题看能不能理解，理解了就快速背一下答案，用关键词记忆法。
  • 哪道题不会了，就去看核心考案上对应的一整个章节，看不懂的去看网课，网课看不懂的 b 站搜索讲解，搜不到的发群里问我。从一道题出发复习一个大知识点，就是所谓的 “以点入面”。
  • 错的题在题号上标注一下，方便二刷的时候知道自己哪里当时曾经错过

总结：这个阶段只做选择题，通过以点入面的方式进行复习，看似在做题，实际上在复习知识点。这个阶段网课和核心考案都是参考资料起到辅助作用，哪里不会就去查一查。主要资料是 1000 题。另外，虽然说要记忆，但是不是像背课文一样死记硬背，没人让你一字不差的默写，重在理解，理解之后留个印象就行。如果这个阶段执行的慢，那就先做马原 + 毛中特，当然最好还是全做完。

# 提分阶段（7-9 月）：闭关修炼到炉火纯青

前面 4 个月的所有工作都是为了今天，从 7 月 1 号开始，进入了最最最关键的时候。别的啥也不说，这三个月做好，70 分保证就有了，重要性不言而喻。

  • 用书： 肖 1000 题 + 疾风劲草 + 徐涛 24 年冲刺班
  • 目标：二刷 1000 题，彻底掌握知识点
  • 策略：以点入面，整理记忆
  • 复习时长：仍然是一定不超过 1.5h

每天 20 道题，答案直接写在书上

  • 做完对答案，进行纠错，错题进行标记，纠错方法同上
  • 狠狠的看疾风劲草，书读百遍，其意自现。没别的，就是把书翻烂
  • 背相应的知识点，关键词记忆法，还是要注意，这里的记忆是为了做选择题，所以重在理解，不是死记硬背
  • 在做题或看书看课过程中，你觉得真正重要的，难记住的知识点，一句话一句话的列举在你的笔记本上 / A4 / 电脑上，标上序号。注意不是抄原话，而是按照自己的理解复述

总结：所以这个阶段过去之后，你会得到一个标注了错题的 1000 题，同时得到了一个笔记本，上面是一句一个知识点，且是自己的理解。后面核心考案不需要了，这个笔记本每周背两次就可以了。

# 定型阶段（10 月）：了然于胸

7-9 是升温，10 月份就是降温，就是去整理归纳和思考，让自己稳定下来。

  • 用书： 王吉掌中宝 + app + 徐涛公众号 + 疾风劲草 + 徐涛 25 年冲刺班
  • 目标：app 上选择题正确率提到 100%
  • 策略：打散学习，查漏补缺
  • 复习时长：2h

重要的是每天学，且不要集中学，建议早上 20min，下午 / 下午 2h。

  • 按计划 10 月 1 号，徐涛会在公众号上出考研带被，b 站可能也有，每天早上六七点发布。你需要起来之后看一看，按照他的计划进行记忆即可。记不住没关系。
  • 把王吉掌中宝翻烂。边看边记，还是那句话，这是为选择题准备的，不需要记得一字不差，重在理解。
  • 使用 app 刷选择题，刷完之后收藏好题和错题，复习对应的知识点。
  • 做点真题的选择题

总结：任务核心还是选择题，做点真题知道自己实力和方向，方便 11 月调整。

# 冲刺阶段（11-12 月）：如鱼得水

走到这一步，反而开始坦然了，因为该做的前面都做好了，现在无非是保持住就行。同时开始背大题。大题背完，75 + 就有了。

  • 用书： 王吉掌中宝 + app + 肖四肖八 + 疾风劲草 + 25 年押题班
  • 复习时长：2h

这一阶段反而更简单了

  • 每天一小时做真题的选择题，然后纠错；
  • 每天一小时背肖四大题，因为 10 月份跟过徐涛带背，所以这里会比较轻松
  • 肖四背完背肖八
  • 空余时间 翻看王吉掌中宝 + 疾风劲草 + 笔记本

# 总结

这个经验贴有点长，但很多细节也没讲清楚，最后我们跳出来再总结一下：

  • 3-4 月就是看课看书，偶尔做做题即可。重在知识的理解和学习。
  • 5-9 月全部都是准备选择题，无论你做什么，都是为了选择题
  • 所以看书看课或者做题的时候，以及背诵选择题知识点的时候，时刻记住不是死记硬背，而是多理解，反复看。
  • 10 月份预热大题，真正背诵就等肖四肖八出来再准备。
  • 选择题要想复习好：坚持每天刷题，有空就看疾风劲草 + 掌中宝，边看边思考，多问自己，这个地方能怎么出题？多使用顺口溜等方法记忆。

等到 12 月份，做一下近两年的真题，我可以组织计时模拟。在后面就是做机构的押题卷，万事俱备，踏踏实实，准备进考场。祝你成功。
