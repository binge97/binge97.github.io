# 1. 使用方法

我建议使用分为两种：

  • 偶尔需要的在线看：比如突然忘了某章节知识，可以找到对应网课，在线观看即可
  • 经常要看的缓存看：比如最近在集中学习政治的马原部分，可以把相应课程直接保存到自己对应的文件夹内，缓存观看。学完及时清理缓存，不然手机没内存了，而且网盘乱。

具体的还是按照自己习惯，怎么方便怎么来。提醒：如果需要保存，记得放在各自文件夹下面。
