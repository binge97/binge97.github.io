# 第二章 工具

“工欲善其事，必先利其器。” 使用工具可以提高效率，过分依赖会浪费时间，所以合理使用，有计划的使用。

# 1.1 视频网站

  • B 站：上面有很多 up 分享，也有网课的上传，也有老师的官方账号，很全面，方便针对性的搜索学习
  • 中国大学生 mooc：大学官方课程，方便基础的学习

# 1.2 搜索工具

  • 知乎：查经验贴，查资讯
  • 小红书：同上
  • chatgpt：一个人工智能问答工具，你可以向它提问任何问题，它都会回答，不一定是正确的答案，但可以作为参考。（账号： angelayqs79@outlook.com 密码：fLfO4k4Tzyh2yM）
  • 微信公众号：很多好的考研公众号

# 1.3 其他

  • notion：就是这个笔记网站，可以用来记录自己学习，制定计划
  • 幕布：思维导图，便于理清计划
  • 百度网盘：可以从微信公众号和闲鱼上买网课资源，会全年同步更新各大机构网课，我记得价格 50 以内
  • 各种 qq 学习群

# 1.4 总结

考研其实就是信息战，网上的资源和信息非常多。使用上面的工具，尝试不同的关键词搜索，然后筛选自己适合的进行借鉴利用。
